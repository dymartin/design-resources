Self-hosted typefaces for the specimen list on pages/fonts.html.

Licensed under the SIL Open Font License 1.1, which permits redistribution.
Files are the latin subset at weight 400, taken from the Google Fonts CDN.

  Inter    Rasmus Andersson    https://fonts.google.com/specimen/Inter

OFL 1.1: https://openfontlicense.org

To add one: download the .woff2 (latin subset), drop it here, add an
@font-face block to assets/css/fonts.css, and add an <li> to the .typefaces
list on the Fonts page.

If a declared face has no file here, the specimen silently falls back — it
will render, just not in the typeface you meant.
